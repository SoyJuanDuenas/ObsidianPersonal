#GameTheory 

Games of cooperation are a type of [[Game]] where players have exactly the same interests, for accomplish this statement the game mush complete the following characteristics:

- No conflict: all players want the same things
- For all $a \in A$ and for all $i,j$ we have $u_i(a) = u_j(a)$
- We often write this games with a single playoff per cell
